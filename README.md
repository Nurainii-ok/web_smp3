
# Website Sekolah

## Deskripsi Project
Website Sekolah merupakan sebuah website informasi sekolah yang dibuat sebagai project pembelajaran dan portfolio.  
Website ini bertujuan untuk menyajikan informasi seputar sekolah secara online dan memiliki sistem pengelolaan konten melalui admin.

Website memiliki dua jenis pengguna, yaitu User dan Admin, dengan hak akses yang berbeda.
 

## Teknologi yang Digunakan
- PHP
- MySQL
- HTML
- CSS
- XAMPP (Local Server)


## Role dan Hak Akses User

### 1. User
User merupakan pengunjung website sekolah.

Hak akses User:
- Mengakses dan melihat seluruh informasi yang tersedia di website
- Melihat halaman seperti profil sekolah, informasi, dan konten lainnya
- User tidak memiliki akses untuk mengubah data


### 2. Admin
Admin bertugas mengelola seluruh isi website.

Hak akses Admin:
- Login ke halaman admin
- Mengelola konten website (CRUD)
- Menambah, mengubah, dan menghapus data
- Mengatur informasi yang ditampilkan pada website sekolah


## Alur Cara Kerja Aplikasi
1. User mengakses website sekolah melalui browser.
2. User dapat melihat seluruh informasi yang tersedia tanpa perlu login.
3. Admin melakukan login untuk mengakses halaman admin.
4. Admin mengelola data dan konten website melalui halaman admin.
5. Perubahan data yang dilakukan admin akan ditampilkan pada website yang diakses oleh user.
